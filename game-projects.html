Project Overview

A narrative-driven horror game concept set in a disturbing neighborhood filled with hidden dangers and psychological tension. The project focuses on atmosphere, exploration, and unsettling character design.

Role

Lead 3D Modeller — Led environment and character modeling, visual asset creation, and consistency of the game's art direction.

Key Features

  • Environment design for horror-focused level layouts

  • Player-focused exploration mechanics

Development Status

Currently in concept and design phase with ongoing asset development and narrative planning.

Technologies Used

Unity Engine, 3ds Max / Blender, Photoshop, Substance

The Last Stop

Project Overview

A technical 3D project focused on developing custom tools in 3ds Max to improve animation, modeling, and rendering workflows. The project emphasizes efficiency through scripting and automation within a full 3D production pipeline.

Role

Lead 3D Modeller — Responsible for asset creation, workflow setup, and coordination between scripting tools and artistic output.

Key Features

  • Custom MaxScript utilities to accelerate modeling and scene management
  • Code-based blending and merging of multiple Mixamo animations
  • Automation tools for lighting setup and scene optimization
  • Asset workflow improvements through scripting support

Development Status

Completed as a module project for 3D scripting and technical art development.

Technologies Used

3ds Max, MaxScript, Mixamo, FBX Workflow
